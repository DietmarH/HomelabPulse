# Security Policy

## Supported Versions

This project is in early development. Security fixes are applied on the default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately via GitHub's built-in vulnerability reporting:
👉 [Report a vulnerability](../../security/advisories/new)

> If you do not have a GitHub account, please contact the maintainer
> directly via the profile link above.

Include:

- Affected component and version/commit
- Reproduction steps or proof of concept
- Potential impact
- Suggested remediation if known

You can expect an initial response within 5 business days.
