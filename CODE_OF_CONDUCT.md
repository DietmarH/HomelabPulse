# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the community

Examples of unacceptable behavior include:

- Harassment, discrimination, or hateful language
- Personal attacks and trolling
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and other contributions that do not align with this Code of Conduct.

## Reporting

Report incidents via [GitHub Discussions](https://github.com/dietmar/HomelabPulse/discussions) or privately through the project maintainer's GitHub profile.
